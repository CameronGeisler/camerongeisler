Definition 2.8.1.
An indexed set of vectors \(\set{\vec{v}_1, \dots, \vec{v}_n}\) in \(\mathbb{R}^n\) is linearly independent if the vector equation,
\begin{equation*} a_1 \vec{v}_1 + \dots + a_n \vec{v}_n = \vec{0} \end{equation*}
has only the trivial solution \(a_1 = \dots = a_n = 0\text{.}\)
