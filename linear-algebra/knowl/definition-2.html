Definition 2.1.3.

A system of linear equations, broadly, is a collection of one or more linear equations involving the same variables. More precisely, a system of \(m\) linear equations in \(n\) unknowns is of the form

\begin{alignat*}{3} a_{11}x_1 \amp + a_{12}x_2 \amp + \dots \amp + a_{1n}x_n \amp = b_1\\ a_{21}x_1 \amp + a_{22}x_2 \amp + \dots \amp + a_{2n}x_n \amp = b_2\\ \vdots \quad \amp \qquad \vdots \amp \amp \qquad \vdots \amp \vdots\\ a_{m1}x_1 \amp + a_{m2}x_2 \amp + \dots \amp + a_{mn}x_n \amp = b_n \end{alignat*}

where \(a_{ij}, b_i \in \mathbb{R}\) are the coefficients of the system. Notice that the coefficients \(a_{ij}\) use double subscript notation. Here, \(a_{ij}\) represents the coefficients of \(x_j\) in the \(i\)th equation.

  • A system of linear equations is also sometimes called simultaneous linear equations, or simply a linear system, or sometimes simply a system.

  • The variables \(x_1, \dots, x_n\) are sometimes called unknowns.

  • A solution of a linear system is an ordered tuple \((s_1, \dots, s_n)\) which, when substituted for \((x_1, \dots, x_n)\text{,}\) respectively, makes each equation a true statement.

  • The set of all possible solutions of a linear system is called its solution set.

  • Two linear systems are equivalent if they share the same solution set. In other words, every solution of the first system is also a solution of the second, and vise versa.
