Theorem 4.1.3. Vector equations as a linear system.

The vector equation,

\begin{equation*} x_1 \vec{a}_1 + \dots + x_n \vec{a}_n = \vec{b} \end{equation*}

has the same solution set as the system of linear equations in \(x_1, \dots, x_n\) which has augmented matrix given by,

\begin{equation*} \begin{bmatrix} \vec{a}_1 \amp \vec{a}_2 \amp \dots \amp \vec{a}_n \amp \vec{b} \end{bmatrix} \end{equation*}
