Theorem 3.1.11. Properties of vector arithmetic.

Let \(\vec{u}, \vec{v}, \vec{w} \in \mathbb{R}^n\) be vectors, \(c, d \in \mathbb{R}\text{.}\) Then,
  1. Addition commutative, \(\vec{u} + \vec{v} = \vec{v} + \vec{u}\text{.}\)
  2. Addition associative, \((\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})\text{.}\)
  3. Additive identity, \(\vec{u} + \vec{0} = \vec{0} + \vec{u} = \vec{u}\text{.}\)
  4. Additive inverse, \(\vec{u} + (-\vec{u}) = \vec{0}\text{.}\)
  5. Scalar multiplication associative, \(c(d\vec{u}) = (cd)\vec{u}\text{.}\)
  6. Scalar multiplication distributes over addition, \(c(\vec{u} + \vec{v}) = c\vec{u} + c\vec{v}\text{.}\)
  7. Vectors distribute over addition, \((c + d)\vec{u} + c\vec{u} + d\vec{u}\text{.}\)
