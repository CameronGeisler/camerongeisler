Theorem 6.2.8. Characterization of linear dependence.

An indexed set of vectors \(S = \set{\vec{v}_1, \dots, \vec{v}_p}\) in \(\mathbb{R}^n\) (where \(p \geq 2\)) is linearly dependent if and only if at least one of the vectors in \(S\) can be written as a linear combination of the others. In other words, there exists a vector \(\vec{v}_i\) that can be written as a linear combination of the other vectors \(\vec{v}_1, \dots, \vec{v}_{i-1}, \vec{v}_{i+1}, \dots, \vec{v}_p\text{.}\) In other words,
\begin{equation*} \vec{v}_i \in \Span{\vec{v}_1, \dots, \vec{v}_{i-1}, \vec{v}_{i+1}, \dots, \vec{v}_p} \end{equation*}
