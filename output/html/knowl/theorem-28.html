Theorem 5.4.9.

The matrices for some rotations include,
\begin{equation*} R_{90^{\circ}} = \begin{bmatrix} 0 \amp -1 \\ 1 \amp 0 \end{bmatrix} \qquad R_{180^{\circ}} = \begin{bmatrix} -1 \amp 0 \\ 0 \amp -1 \end{bmatrix} \qquad R_{270^{\circ}} = \begin{bmatrix} 0 \amp 1 \\ -1 \amp 0 \end{bmatrix} \end{equation*}
