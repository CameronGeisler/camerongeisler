Definition 8.1.1.

A vector space is a non-empty set \(V\) of objects, called vectors, on which are defined two operations, called addition and scalar multiplication by scalars (real numbers), that satisfy the following axioms. For all vectors \(\vec{u}, \vec{v}, \vec{w} \in V\text{,}\) and all scalars \(c, d \in \mathbb{R}\text{,}\)

  1. Closure under addition and scalar multiplication, the sum of \(\vec{u}\) and \(\vec{v}\text{,}\) \(\vec{u} + \vec{v}\text{,}\) is in \(V\text{,}\) and the scalar multiple of \(\vec{u}\text{,}\) \(c\vec{u}\text{,}\) is in \(V\)

  2. Associativity, \((\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})\text{.}\)

  3. Commutativity, \(\vec{u} + \vec{v} = \vec{v} + \vec{u}\text{.}\)

  4. Additive identity, there exists a zero vector \(\vec{0}\) such that \(\vec{u} + \vec{0} = \vec{u}\text{.}\)

  5. Additive inverse, for every \(\vec{u} \in V\text{,}\) there exists a vector \(-\vec{u} \in V\) such that \(\vec{u} + (-\vec{u}) = \vec{0}\text{.}\)

  6. Associativity of scalar multiplication, \(c(d\vec{u}) = (cd) \vec{u}\text{.}\)

  7. Scalar multiplication distributes over addition, \(c(\vec{u} + \vec{v}) = c\vec{u} + c \vec{v}\text{.}\)

  8. Vector distributes over scalar addition, \((c + d) \vec{u} = c\vec{u} + d \vec{u}\text{.}\)

  9. Multiplicative identity, \(1 \vec{u} = \vec{u}\text{.}\)
