Let \(A\vec{x} = \vec{b}\) be a linear system, which is consistent with solution \(\vec{v}_p\text{.}\) Then, the solution set of \(A\vec{x} = \vec{b}\) is the set of all vectors of the form \(\vec{v} = \vec{v}_p + \vec{v}_h\text{,}\) where \(\vec{v}_h\) is any solution of the associated homogeneous system.
