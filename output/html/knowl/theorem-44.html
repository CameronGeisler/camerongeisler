Theorem 6.2.7. Linear independence as a homogeneous system.

Vectors \(\vec{v}_1, \dots, \vec{v}_p\) are linearly independent if and only if for the matrix \(A = \begin{bmatrix} \vec{v}_1 \amp \dots \amp \vec{v}_p \end{bmatrix}\text{,}\) the homogeneous system \(A\vec{x} = \vec{0}\) has only the trivial solution. Equivalently, if and only if \(N(A) = \set{\vec{0}}\text{.}\)
