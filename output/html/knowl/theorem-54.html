Theorem 7.3.1. Determinants and elementary row operations.

Let \(A\) be a square matrix, \(B\) be the matrix resulting from \(A\) after a single row operation. Then,
  1. If a multiple of row row is added to another row, then \(\det{B} = \det{A}\text{.}\)
  2. If two rows are interchanged, then \(\det{B} = -\det{A}\text{.}\)
  3. If one row is scaled by \(k \neq 0\text{,}\) then \(\det{B} = k \cdot \det{A}\text{.}\)
